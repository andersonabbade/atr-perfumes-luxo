# 🖤 ATR Imports - Perfumes de Luxo

Landing page moderna para uma loja de perfumes importados, desenvolvida com foco em **design premium, experiência do usuário e conversão de vendas**.

---

## 🚀 Demonstração

🔗 Acesse o site:
👉 https://atr-imports.netlify.app

## ✨ Funcionalidades

* 🛍️ Catálogo de perfumes organizado por categorias
* 🔍 Filtro de produtos (Masculino, Feminino, Árabe, etc)
* 📱 Layout responsivo (mobile e desktop)
* ⚡ Interface rápida e moderna
* 📲 Integração com WhatsApp para contato direto
* 🎯 Foco em conversão (design voltado para vendas)

---

## 🛠️ Tecnologias utilizadas

* React
* Vite
* JavaScript
* CSS3

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com foco em:

* Criar uma **landing page profissional para clientes reais**
* Praticar **React na construção de interfaces modernas**
* Trabalhar com **componentização e organização de código**
* Aplicar conceitos de **UX/UI voltados para vendas**
